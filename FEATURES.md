# The Black Gold Roller Coaster - Feature Overview

## Design Philosophy

This website is built with a **modern, clean aesthetic** using:
- **Pastel Color Palette**: Soft rose, amber, blue, green, and purple tones
- **No Dark Mode**: Bright, premium backgrounds throughout
- **Smooth Animations**: Native CSS transitions (no animation libraries needed)
- **High Interactivity**: Every element responds to user input
- **Mobile Responsive**: Beautiful on all devices

## Color System

### Primary Palette (Pastel Tones)
- **Rose**: #fca5a5, #fed7aa
- **Orange**: #fed7aa, #fbbf24
- **Yellow**: #fef08a, #ffeb99
- **Blue**: #bfdbfe, #87ceeb
- **Purple**: #d8b4fe, #c084fc
- **Green**: #bbf7d0, #86efac
- **Pink**: #fbcfe8, #fca5a5

### Background Colors
- **Sections**: White or light pastels (slate-50, blue-50, emerald-50)
- **Text**: Slate-700 to slate-900 (dark, readable)
- **Borders**: 2px solid, matching section theme

### Animation Palette
- **Hover**: Scale 105%, enhanced shadow
- **Active**: Full color highlight, scale 105%
- **Transitions**: 300-500ms smooth easing
- **Loading**: Fade in on scroll

---

## Interactive Patterns

### Button Interactions
```
Default: bg-slate-200, text-slate-700, border-transparent
Hover: bg-slate-300, scale(1.05)
Active: bg-gradient (color-specific), text-white, scale(1.05), shadow-lg
```

### Card Interactions
```
Default: bg-white/slate-50, border-2 (color-200), shadow-xl
Hover: shadow-2xl, scale(1.05), border brightens
Active: bg (color-100), border brightens, scale(1.05)
```

### Bar Charts
```
Container: bg-slate-200, rounded-full, border-2 border-slate-300
Bar: Pastel color, smooth width animation (500ms)
Hover: container border brightens
```

## Performance Optimizations

- **Lazy 3D Rendering**: Three.js models only render when visible
- **CSS Animations**: Hardware-accelerated transforms
- **Smooth Scroll**: Native browser scroll behavior
- **Intersection Observer**: Efficient scroll detection
- **Optimized Assets**: Minified CSS (32.74 kB gzip)
- **Code Splitting**: Separate modules for each chapter

---

## Accessibility

- Large readable text (18px base, 24-36px headings)
- High contrast ratios (dark text on light backgrounds)
- Emoji visual aids alongside text
- Clear interactive targets (minimum 44x44px)
- Semantic HTML structure
- Focus states on all interactive elements
